About CosmicPi

The Cosmic Pi project aims to build the world's largest open source distributed cosmic ray telescope. You too can be a part of the project, by becoming a Cosmic Pixel!

The CosmicPi software is licenced under the GNU GENERAL PUBLIC LICENSE Version 2 or later. You can receive a copy of the licence here. For the source code of the CosmicPi software please visit the projects github page.

This Cosmic Pi will automatically publish all cosmic ray and associated meta-data (position, accelerometer, magnetometer, temperature, humidity, pressure) to the internet for anyone to use under a CC0 license (no rights reserved) and placed in the public domain.
